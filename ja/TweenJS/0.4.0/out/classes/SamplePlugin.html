SamplePlugin Class

Defined in: src/tweenjs/SamplePlugin.js:33
Module: TweenJS

TweenJSプラグインのサンプル。このプラグインは、実際には決してトゥイーンに影響を与えません。単にTweenJSプラグインの 作り方を記録することを意図するものです。インライン・コメントのためにコードを見てください。

TweenJSプラグインは単に、１つのプロパティ(priority)と３つのメソッド (init, step, and tween)を公開しているだけのオブジェクトです。 通常、プラグインは同様にinstallメソッドも公開しています。けれども、これは厳密には必要ありません。

Defined in src/tweenjs/SamplePlugin.js:66

このプラグインが登録される新しいトゥイーンプロパティが初期化するときにTweenJSによって呼ばれます。 一般的に、Plugin.init呼び出しは、Plugin.toに続いて即座に行われます。

Parameters:

  • tween Tween

    関連するトゥイーンインスタンス。

  • prop String

    初期化されるプロパティの名前。

  • value Any

    トゥインのターゲット上のプロパティの現在値。

Returns:

Any:

プロパティの開始トゥイーン値。多くの場合、単に値のパラメータを返します。 しかし、いくつかのプラグインは開始値の変更が必要になるかもしれません。

install

() static

Defined in src/tweenjs/SamplePlugin.js:55

TweenJSと一緒に使用するために、このプラグインをインストールします。そして、このプラグインが扱うプロパティのリストを 登録します。このプラグインを有効にするために、TweenJSがロードされた後で一度、これを呼び出してください。

Any static

Defined in src/tweenjs/SamplePlugin.js:99

このプラグインが登録されるトゥイーンプロパティを進めるときに呼び出されます。

Parameters:

  • tween Tween

    関連するトゥイーンインスタンス。

  • prop String

    トゥイーンされるプロパティの名前。

  • value Any

    TweenJSによって計算された、プロパティの現在のトゥイーンされた値。

  • startValues Object

    現在のステップの始まりにおけるすべてのプロパティのハッシュ値。startValues[prop]を 使うことで現在のプロパティの開始値にアクセスできます。

  • endValues Object

    現在のステップの終わりにおけるすべてのプロパティのハッシュ値。

  • ratio Number

    現在のステップでのイースのプログレスを示す値。この数値は通常0と1の間になります。 けれども、いくつかのイースはこの範囲外の値を生成します。

  • wait Boolean

    現在のステップが"wait"ステップかどうかを示します。

  • end Boolean

    トゥイーンが終わりに到達したかどうかを示します。

Returns:

Any:

ターゲットプロパティに代入された値を返します。たとえば、Math.round(value)は、 整数としての計算値を代入します。Tween.IGNOREは、ターゲットプロパティに値を代入するのを防ぎます。
