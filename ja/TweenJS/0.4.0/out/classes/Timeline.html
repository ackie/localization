Timeline Class

Defined in: src/tweenjs/Timeline.js:34
Module: TweenJS

Timelineクラスは、複数のトゥイーンを同期し、１つのグループとして管理するようにします。 タイムラインがループしている場合は、トゥイーンの"loop"プロパティがfalseになっていてもタイムライン上のトゥイーンが ループするようにみえる場合があることに注意してください。

Parameters:

  • tweens Object

    このタイムラインに追加するトゥイーンの配列。詳しくは addTweenを見てください。

  • labels #crossLink "Timeline/setLabels"

    gotoAndPlay/Stop用のラベルを定義するオブジェクト。 詳細は }{{/crossLink}}を見てください。

  • props Loop:true

    このトゥイーンインスタンスに適用する構成プロパティ (ex. )。 すべてのプロパティのディフォルトは falseとなります。 サポートされるプロパティは次のとおり:

    • loop: このトゥイーンにloopプロパティを設定します。
    • useTicks: すべての時間にミリ秒ではなく tickを使用します。
    • ignoreGlobalPause: このトゥイーンに ignoreGlobalPause を設定します。
    • paused: 一時停止されたトゥイーンが開始するかどうかを示します。
    • position: このタイムラインの最初の位置を示します。
    • onChanged: このタイムラインの onChange ハンドラを示します。
