AlphaMapFilter Class

Extends Filter
Defined in: src/easeljs/filters/AlphaMapFilter.js:34
Module: EaselJS

グレースケールのアルファマップ画像 （又は canvas） を対象に適用します。その際、マップの赤のチャネルが結果のアルファチャネルに コピーされます。RGB チャネルは対象からコピーされます。

一般的には、パフォーマンす上の観点から AlphaMaskFilter の使用が推奨されます。

フィルターの使い方の例は Filter を参照して下さい。
