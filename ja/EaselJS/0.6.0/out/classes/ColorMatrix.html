ColorMatrix Class

Extends Array
Defined in: src/easeljs/filters/ColorMatrix.js:34
Module: EaselJS

ColorMatrixFilter で使用する行列の生成を支援する関数を提供します。 もしくは、直接 ColorMatrixFilter 用の行列として使用できます。ほとんどのメソッドは連続した呼び出しを 容易にするようインスタンスを返します。

例

myColorMatrix.adjustHue(20).adjustBrightness(50);

フィルターの使い方の例は Filter を参照して下さい。
