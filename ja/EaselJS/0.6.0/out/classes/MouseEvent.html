MouseEvent Class

Defined in: src/easeljs/events/MouseEvent.js:34

onMouseMove

Function deprecated

Defined in src/easeljs/events/MouseEvent.js:122

Deprecated: "mousemove" イベントの追加により、将来のバージョンでは廃止される予定です。

"onPress" イベントの場合のみ、onMouseMove にハンドラーを指定できます。 このハンドラーは、マウスが解放されるまで、マウスが移動すると毎回呼ばれます。 これは、ドラッグ & ドロップのような操作に便利です。

Events

mousemove

Defined in src/easeljs/events/MouseEvent.js:57

Available since 0.6.0

"mousedown" の MouseEvent オブジェクトからは、ユーザーがマウスを放すまで mousemove イベントが発行されます。 これにより、マウス押下中のマウスの移動を知ることができ、ドラッグ & ドロップ等の実現に役立ちます。 イベントのプロパティについては MouseEvent クラスを参照して下さい。

mouseup

Defined in src/easeljs/events/MouseEvent.js:65

Available since 0.6.0

"mousedown" の MouseEvent オブジェクトからは、ユーザーがマウスを放すと mouseup イベントが発行されます。 これにより、特定のマウス押下に対応するマウス解放を知ることができ、ドラッグ & ドロップ等の実現に役立ちます。 イベントのプロパティについては MouseEvent クラスを参照して下さい。
